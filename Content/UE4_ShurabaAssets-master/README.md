# 修羅場アセット

きっといつか役に立つかもしれないくらいのUE4ユーザアセット。  
ご自由に使うなり改造して頂いても構いません。  


## BF_ResolutionConfig.uasset

解像度関係のBP関数が定義されています。  

・Set Client Resolution  
-> 解像度及びフルスクリーン設定  
・Check Supported Fullscreen Resolution  
-> 指定解像度が実行環境で対応しているかを確認する関数  
・Print Supported Fullscreen Resolutions  
-> 実行環境の解像度一覧を表示するDebug関数  



# 導入手順

「Content」フォルダ毎導入したいプロジェクトのフォルダにコピーすれば使えます。  

動作確認環境は「UE4.10」となります。  
APIの変更がなければそれ以上のエンジンVerであれば実行可能かもしれません。
